Document Objects:-
	The window document object represents the document objects that is currently loaded in the window. A document holds information like the document properties like html, the forms collection, string, and its location. It also implements a different interface.

	The document object represents the whole html document as a tree of Objects like html, head, body, and their tags. It represents the html document.

	Each HTML document that gets loaded into a window becomes a document object. The document contains the contents of the page. Using document object, JavaScript can modify, add and delete the HTML elements, attributes CSS styles in the page. You can access a document object either using window. Document property.

	Document Object represents the document loaded inside the window or browser.

	We can not access windows objects properties inside the document object.

	Document object is the root element of the document object model.

	Examples are, document.close() , document.write() , document.open()

Window Objects:-
	Window Object contains objects that represent the current window. It holds things like window location, window history, window screen, window status, or the window document. 

	It has information about the framing setup like the frames, parent, top, self-properties, and holds important interfaces such as application Cache, HttpRequest, console.

	The window object has a property called document which stores a representation of the document such as title and anchors The window object represents the browser window where your document is displayed.

	It represents the browser window in which you are seeing the content.

	We can access document object properties

	The window is the global element for all objects, functions.

	Example , Alert(), prompt(),open().
